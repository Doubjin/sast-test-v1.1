## 🎬 화면 전환 스펙 — 3D 입체 전환

> **컨셉**: 카드들이 실제 3D 공간에 존재하는 것처럼 깊이감 있게 이동

### 질문 간 전환 (메인 전환 방식)

```
이전 질문 카드:
  → translateZ(0) → translateZ(-80px) + scale(0.92) + opacity(1→0)
  → duration: 350ms, easing: cubic-bezier(0.4, 0, 0.2, 1)

다음 질문 카드:
  → translateZ(80px) + scale(0.92) + opacity(0) → translateZ(0) + scale(1) + opacity(1)
  → duration: 350ms, easing: cubic-bezier(0.4, 0, 0.2, 1)
  → delay: 100ms
```

### 섹션 간 전환 (PART 전환, 결과 진입)

```
카드 뭉치 전체:
  → 3D rotateX(5deg) + translateY(30px) + opacity(0) 로 사라짐
  → 새 섹션: rotateX(-5deg) + translateY(-30px) 에서 진입
  → duration: 500ms, perspective: 1200px
  → 배경 그라데이션도 함께 morphing
```

### 답변 버튼 인터랙션
```
선택 시:
  → scale(1) → scale(0.88) → scale(1.08) → scale(1)  (젤리 Bounce)
  → duration: 300ms, spring 계열 easing
  → 내부 색상 fill: 0% → 100%, ripple 효과 확산
  → 선택 완료 후 600ms delay → 자동으로 다음 화면 전환
```

### 진행 바 애니메이션
```
답변 완료 시:
  → progress bar: width animation, ease-out
  → 끝단 dot이 살짝 bounce하며 도달
  → 50%, 100% 달성 시 미세한 confetti 파티클 연출
```
